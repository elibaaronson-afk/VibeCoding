Portfolio landing page template

Files created:
- index.html — responsive landing page outline (replace placeholders with your info)
- css/styles.css — minimal responsive styles
- js/main.js — small interactive helpers (mobile nav, year)
- assets/ — place profile images and other media here (use modern formats + srcset)

Usage:
Open index.html in a browser. Replace placeholders and add your images to assets/. For a production site, connect a form handler or use a static-site hosting service.
